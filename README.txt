COMO USAR

1. Extraia os arquivos na sua pasta do projeto:
   C:\Users\ciril\OneDrive\Documentos\03 - Reserva da Serra\05 - CHATBOT

2. Renomeie .env.example para .env

3. Preencha no .env:
   SUPABASE_SERVICE_ROLE_KEY
   OPENAI_API_KEY
   WHATSAPP_ACCESS_TOKEN
   WHATSAPP_PHONE_NUMBER_ID

4. Abra o terminal na pasta do projeto e rode:
   npm install

5. Rode o servidor:
   npm run dev

6. Teste no navegador:
   http://localhost:3000
